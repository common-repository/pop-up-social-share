== Description ==

Pop Up Social Share is Simple Light Social Plugin using the default WordPress Jquery and Compatible upto WordPress Version 4.5.2. We are still working on it to have more features and more social option , although it will work perfectly for the facebook , twitter , pinterest and Google Plus.



**Supported Social Shares**

* Facebook
* Twitter
* Pinterest
* Google+
* Email

== Installation ==
1. Unzip the `popup-social-share.zip`.
2. Upload the file to the `/wp-content/plugins/` directory.
3. Activate the plugin through the `Plugins` menu in WordPress.
4. Configure the options under Admin Panel `Settings -> PopUp Share`.
5. Do Necessary Setings

== Frequently Asked Questions ==

= Why to use this plugin ? =

This plugin is light weight , with addition of Pop Up so that your user will stay in same page, No more third party script needed to load in your site.

= Can I change the size of the social icon ? =
Yes , if you know the little CSS , you can overwrite the existing one 

= Does my theme support Pop Up Social Share ? =
Yes, No image or font has been used ,so it will support any theme
